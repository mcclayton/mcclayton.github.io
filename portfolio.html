Media Matrix

Oct. 17, 2014 – Oct. 19, 2014
Team Members: Michael Clayton, Sam Dickson, Niraj Venkat

Language(s): Java, XML
Tools and Libraries: Android SDK, Firebase

Developed an Android application at the Purdue BoilerMake Hackathon, which enables users to create one large screen out of their collective mobile devices. Media Matrix uses a custom bin-packing algorithm to optimally arrange the devices in such a way that view space is maximized. Images are cast to the devices in real-time from a parent 'remote' device.

Map Viewer and Map Editor

March 25, 2013 - April 22, 2013

Language(s): Java

Also has support for disconnected graphs.
