## View Live Site
http://www.michaelcclayton.com

## Run Locally
The development server can be started locally with:
```
yarn start
```

## Run Tests
```
yarn test
```

## Deploy Site
To deploy this site to [Github Pages](https://pages.github.com/) (must be properly authenticated), run:
```
yarn deploy
```
